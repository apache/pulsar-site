Client Release Notes‚Äã

Tip

  • The code repos of Java and WebSocket clients are hosted in the Pulsar main repo. These clients are released with Pulsar.

  • The code repos of C++, Python, Go, Node.js, and C# clients are hosted outside of the Pulsar main repo. These clients are not released with Pulsar.

  • All clients have independent release notes and they are automatically synced to this page.
