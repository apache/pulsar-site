◆ setCompressionType()

ProducerConfiguration& pulsar::ProducerConfiguration::setCompressionType ( CompressionType  compressionType)

Set the compression type for the producer.

By default, message payloads are not compressed. Supported compression types are:

  • CompressionNone: No compression
  • CompressionLZ4: LZ4 Compression https://lz4.github.io/lz4/
  • CompressionZLib: ZLib Compression http://zlib.net/
  • CompressionZSTD: Zstandard Compression https://facebook.github.io/zstd/ (Since Pulsar 2.3. Zstd cannot be used if consumer applications are not in version >= 2.3 as well)
