Pulsar admin CLI

The pulsar-admin tool enables you to manage Pulsar installations, including clusters, brokers, namespaces, tenants, and more.

Usage


$ pulsar-admin command

  • broker-stats
  • brokers
  • clusters
  • functions
  • namespaces
  • ns-isolation-policy
  • sink
  • source
  • topics
  • tenants
  • resource-quotas
  • schemas

Options

--limit 2G \
--policy producer_request_hold
