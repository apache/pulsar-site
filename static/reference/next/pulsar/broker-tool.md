# broker-tool



```shell
$ pulsar broker-tool subcommand
```



## <em>load-report</em>

```shell
$ pulsar broker-tool load-report options
```

|Flag|Description|Default|
|---|---|---|
| `-h, --help` | Display help information|false|
| `-i, --interval-ms` | Interval to collect load report, in milliseconds|100|
