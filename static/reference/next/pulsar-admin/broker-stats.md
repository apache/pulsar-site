# broker-stats

## monitoring-metrics

dump metrics for Monitoring

**Command:**

```shell
$ pulsar-admin broker-stats monitoring-metrics options
```

|Flag|Description|Default|
|---|---|---|
| `[-h, --help]` | Show this help message and exit.|null||
