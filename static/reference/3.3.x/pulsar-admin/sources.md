# sources

Interface for managing Pulsar IO Sources (ingress data into Pulsar)


```shell
$ pulsar-admin sources subcommand
```



## create

Submit a Pulsar IO source connector to run in a Pulsar cluster

**Command:**

```shell
$ pulsar-admin sources create options
```
